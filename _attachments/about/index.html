The 0P3NR3P0 is an open/public repository of glitch art worx

a modular art/archive project developed as an open port for anyone anywhere to submit glitch art worx that can be represented by a URL ( i.e. video, images, sound, web ). Artists interested in adding a work to the repo can submit here + will recieve a confirmation URL ( which includes an edit URL to edit the entry in the future ). One step process; no usernames + no passwords. Folks interested in searching through the repo can do so using the search on the home page.


IRL/exhibition history

  • initially the 0P3NR3P0 served as a remote submission system for the 2112 GLI.TC/H festival/conference/gathering in Chicago.
  • the second IRL exhibition was held on March 19, 2013 at the Museum of Contemporary Art in Chicago
  • the third IRL exhibition was held at Furtherfield Gallery in London, from June 8 - July 28, 2013
  • the fourth IRL installation focused specifically on political glitch worx for the tacticalgl.it/ches/ show at SUDLAB in Naples, Italy.
  • the fifth IRL installation will be held at the Ukrainian Institute of Modern Art as part of glitChicago, to include your work visit the submit page here
  • IRL/online galleries interested in leveraging the repo for a show visit the curate page.

We beleive that "access drives preservation", for this reason the 0p3nr3p0 is an entirely open-source project ( on github ) + a copy of the repo's entire database of work-linx ( w/meta-data ) is perpetually available for download in .csv and .json format:
